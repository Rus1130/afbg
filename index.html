How to add video

  1. If the folder does not exist already, create it. Make it an abbreviated name, or an old name, something short, like "katana" or "k23" for "Prototype-K23".
  2. Inside of that folder, you would put the mp4 file that is going to be used inside of the wiki template.
    • For normal moves, it should be the key to press in order to use that move. So kickboxers first move, Neck Grip, would be 1.mp4 inside of the kb folder.
    • For ults, prefix "u" to the filename. so steel's first ult move would be u1.mp4
    • For variant moves, suffix with "v". For example, kickboxers aerial variant for move 4 would become 4v.mp4
    • For finishers, suffix with "f". For example, the finisher for greatsword's move 4 would become 4f.mp4
    • For single-use ultimates, the filename should be ultimate.mp4
  3. To reference the video inside of the Wiki, you must use this template:
    {{Video|name=FOLDER_NAME|move=FILE_NAME}}
    So for kickboxers move Neck Grip, you would use:
    {{Video|name=kb|move=1}}
